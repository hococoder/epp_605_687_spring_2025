Getting Ready

In your eighth programming assignment, you'll add video and In App Purchase support to the app. Please download the starter project from the following link to get started:

Homework 8 Starter (link will be active start of week 8)

Assignment Requirements

For this week's homework, please do the following:

  • In the Assignment View, add in a new section to display a video for the assignment. To make things simple, you can use the Video URL from the example in the AVKit module for all assignments.
  • In the About View, add 2 things:
    • A view that represents an ad in the app
    • A button that disables the ad via an In App Purchase
      • Make sure to use the techniques we discussed in the lecture to setup sample In App Purchases in your project (via the StoreKit Configuration File), so that your app can exercise the purchase when the button is tapped and the purchase takes place.
      • Make sure that when the app is shutdown, and then started again, the app recognizes that the IAP has been purchased, and keeps the ad hidden!

Assignment Submission

  • Zip up the Xcode project folder. This should be the folder that has the .xcodeproj file inside it.
  • Rename the zip to LastName_Homework#.zip
  • Upload the zip to the Programming Assignment 8 area on Blackboard BEFORE 11:59 PM ET on the day it is due (Sunday of Week 8).
  • Be sure to test your app before submitting, so you know it works as intended!

Assignment Rubrics

In addition to the normal grading criteria stated in the syllabus, you will be graded on:

  • The video in the assignments view should play properly.
  • The IAP should disable the ad for the given session and every start of the app after that.
