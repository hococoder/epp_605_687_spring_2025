Getting Ready

In your 11th programming assignment, you'll add some shortcut capabilities and ask the user for feedback. To get started, download the zip below and get the starter project for Homework 11.

HW 11 Starter (link will be active start of week 11)

Assignment Requirements

For this week's homework, do the following:

  • Find a place to ask for user feedback in the app. It's up to you where it takes place, but make sure that it doesn't immediately ask for feedback - ensure a certain amount of interaction with the app before it asks for feedback. Please tell me how to activate your request for feedback in the Blackboard submission details.
  • Add support for shortcuts into the Assignments. Make sure that:
    • Shortcuts are established for each assignment and can be differentiated! (Meaning that if I go to assignments 1 and 2, they should both appear as separate shortcuts)
    • The shortcuts are donated appropriately (see note below)
    • The shortcuts appear in the Search area in iOS once donated (drag down on the home screen)
    • The shortcuts navigate to the proper assignment once tapped.

Here are some things to keep in mind for this homework:

- To properly test the shortcut taking to you to the correct assignment, make sure the assignment table is showing before you hit the home button to send the app to the background. If you don't, the navigation as we set it up in class won't work.
- Make sure your bundle ID is set properly in the all the necessary locations when adding the NSUserActivity.

Assignment Submission

  • Zip up the Xcode project folder. This should be the folder that has the .xcodeproj file inside it.
  • Rename the zip to LastName_Homework#.zip
  • Upload the zip to the Programming Assignment 6 area on Blackboard BEFORE 11:59 PM ET on the day it is due (Sunday of Week 11).
  • Be sure to test your app before submitting, so you know it works as intended!
