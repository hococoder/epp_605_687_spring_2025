Getting Ready

In your fourth programming assignment, you'll use network calls to retrieve data and use that data to populate your app's local database. To get started, download the materials (with the link below) if you haven't already:

Downlink HW 4 Starter (link will be active start of week 4)

Don't forget to rename the project so it's easier for me to track which project is which on the simulator!

Assignment Requirements

In this assignment, you're going to extend the reach of your app to the network to not only load but save data. Do the following:

  • Instead of depending on JSON files built into the project, you will need to update your project to reach out to the following URLs to get JSON data for each of the tables:
    • Students: http://159.203.191.136/resources/students.json
    • Assignments: http://159.203.191.136/resources/assignments.json
    • Teachers: http://159.203.191.136/resources/teachers.json
    • Courses: http://159.203.191.136/resources/courses.json

When you reach out to grab data from the URLs, you must use async/await!

A method using generics in your ClassAssignmentsCoreDataModel may come in handy here!

  • Add a screen between the login screen and the content view (the 4 tabs) that displays that the network resources are loading. REMEMBER: Your network calls are asynchronous so there is NO guarantee what order they will come back in - so you should make sure all data has been downloaded before moving onto the main content view.
  • Update your app to use the proper iCloud container (using your JHED) in the project settings. You will need to hover over the list of containers with your mouse to find out the correct one to choose (the one that ends with your JHED). Unselect all others.
  • Add a "Submit a Review" button to the Teacher page,:
    • When pressed, have it popup a sheet that allows the user to leave a rating (out of 5) and a text review.
    • The Sheet that pops up should have a "Done" button on the right side of the toolbar, and a "Cancel" button on the left.
    • Cancel should dismiss the dialog
    • Done should collect the information from the review, make a new CKRecord for the review, and save it to the private database in CloudKit.

Assignment Submission

  • Zip up the Xcode project folder. This should be the folder that has the .xcodeproj file inside it.
  • Rename the zip to LastName_Homework#.zip
  • Upload the zip to the Programming Assignment 4 area on Blackboard BEFORE 11:59 PM ET on the day it is due (Sunday of Week 4).
  • Be sure to test your app before submitting, so you know it works as intended!

Assignment Rubrics

  • Async/await should be used to grab data from the network
  • A "loading data from network" screen should now appear in the app after login
  • Your project should have the proper CloudKit container added
  • The teacher page should now have a "Submit a Review" button with the requirements listed above, and that should get submitted to CloudKit when the user clicks "Done"
