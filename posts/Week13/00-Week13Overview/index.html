Welcome to Week 13! This week, you'll learn about a few important Apple-side tools for submitting your app to the app store, as well as take a look back at this year's WWDC!

  • Introduction to User Feedback - Learn about how to ask for user feedback right from your app
  • Introduction to Developer Portal - Learn how to make certificates, provisioning profiles and more (just in case Xcode doesn't get it quite right)
  • Introduction to App Store Connect - Learn how to setup metadata about your app, get sales information and more!
  • Introduction to TestFlight - Learn all about Apple's in house testing service, TestFlight
  • WWDC Recap - There was a lot of new things that came out at WWDC this past summer, so in this module we'll touch on a few areas you can explore in your future iOS learning.

There is no homework assignment this week - finish up your final projects!

There is NOT a Week13_Starter or Week13_Final project for this week; example code can be found in the EPExampleApp project in the resource bundle
