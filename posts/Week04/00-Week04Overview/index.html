Downlink Week 4 Materials

Welcome to Week 4! This week we'll delve into one of the most important framework for working with mobile apps - Networking!

  • Introduction to Networking: In this module, we'll go over how to communicate with remote systems with URLSession and its family of associated classes, delegates and protocols
  • Introduction to Swift Package Manager: Sometimes it's best not to reinvent the wheel - Swift Package Manager lets developers easily load in external libraries into their project
  • Async/Await - this is new in iOS 15, and greatly decreases the amount of networking code you'll write!
  • Syncing with CloudKit: Apple provides a "database in the cloud" experience for storing data - we'll see how easy it is to setup and work with
  • Programming Assignment 4: In this assignment, you'll build networking functionality into the app so it can communicate with the class server and download assets and other files.
  • Development Log 3: This is it! You've gathered your requirements, sketched some UI, and thought about data flow in your app. In this Development Log, you'll define the final set of tasks in JIRA to include in your final project.

There is NOT a Week4_Starter or Week4_Final project for this week; example code can be found in the EPExampleApp or in the CloudKitTest project in the resource bundle. There is, as usual, a starter project for the homework
