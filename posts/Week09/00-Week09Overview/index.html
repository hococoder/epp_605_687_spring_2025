Welcome to Week 9! This week you'll focus entirely on HealthKIt - the framework for inputting, gathering and viewing data on your iOS device.

  • Introduction to HealthKit - learn all about Apple's built in framework for health monitoring
  • Programming Assignment 9 - You'll also add a space for students to do a health self-checkin right from the app.

There is NOT a Week9_Starter or Week9_Final project for this week; example code can be found in the EPExampleApp project in the resource bundle. There is, as usual, a starter project for the homework
