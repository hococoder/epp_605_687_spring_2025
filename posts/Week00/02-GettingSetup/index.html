Downloading and Installing Xcode

Getting Xcode is easy – simply go to the App Store to download it. You’ll need an Apple ID, which you probably have already).

  • All projects require Xcode 16.2 or ABOVE, which comes pre-installed with Swift 6.0
  • You also need to be running on macOS 15.2 (macOS Sequoia) or above.

Swift

We will use Swift in this class – that’s where Apple’s heading in terms of language support, so we’ll spend our time here. The version of Swift we’ll use is 6.0 – and is included in the download of Xcode, so no extra installation is needed!

We won’t be using Objective C in this course (I consider it a “heritage language”), but be aware that you’ll find many examples for iOS that are in Objective C. Most examples will translate to Swift, but others may not! The more popular Stack Overflow examples tend to get updated for the various Swift updates, so keep an eye out for the right version.

Programming Assignments

All projects must:

  • Use the version of Xcode we are currently using at that point in the semester.
  • Must not use third party tools like Appcelerator. If I can’t load the project into Xcode and successfully compile and run it (or if I find evidence you have used a non-Xcode IDE), you will get a zero (0) on the assignment.
  • Target iOS 18.2 or above (whatever we are using at that point in the semester)
  • Run on a system running the macOS 15.2 (Sequoia) or above.

There is no hardware requirement – we’ll develop and run everything on the simulator - but you may find having your device available to run demos on or write your final project against useful. The next submodule discusses how to setup a device for testing, in case you are interested. A zip of your project folder will be submitted for programming assignments.

Assignment Submission

  • Submissions of all Homework assignments will occur through links in Canvas
  • Development Log Assignments will occur in JIRA (I can see when they are submitted!)
  • Peer review assignments will occur through Canvas links
  • Final projects will occur through Canvas links

For any assignments that may be too large, links to Sharepoint or something similar will be provided on an as-needed basis.

DON'T UPGRADE YOUR TOOLS UNLESS OTHERWISE INSTRUCTED

Don’t upgrade your tools mid-semester UNLESS INSTRUCTED! This can cause problems if APIs change. If you change versions when you weren't told to, your code may not compile and your grade may suffer. If you happen to need a previous version of Xcode, you can get it at developer.apple.com.

If you do end up using your device for developing your final project, be sure to keep the iOS version the same as the one required for this class!
