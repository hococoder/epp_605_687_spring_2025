Welcome to Week 2! This week we'll cover 2 of the newest technologies to be introduced into the Swift ecosystem: Combine and SwiftUI.

  • Intro to Combine: Combine is Apple's solution to the "reactive" technology craze of late, seen in libraries such as React. It is a declarative Swift API for processing values over time, and can include asynchronous events. This makes it great for updating user interface elements as data becomes available. Speaking of which...
  • Intro to SwiftUI: SwiftUI is Apple's next big leap forward in making user interfaces via declarative programming. Gone are the days of making objects and trying to figure out exactly where they need to go on a storyboard. SwiftUI allows you to declare that elements should be created, should have a particular color or size (or various other attributes), and the layout system worries about how to render it on the device you're running the app on - obeying the built in human interface guidelines for you, which is really nice.
  • Xcode Tips and Tricks: Learning SwiftUI introduces a lot of interaction with the broader Xcode interface, so we'll take a tour of some of the features of Xcode along with some tips and tricks
  • Programming Assignment 2: Last week, you defined the model - now it's time to flesh out the user interface!
  • Development Log 2: Speaking of user interfaces - how will the user interface in your app look? How will data flow from screen to screen? In this log, you'll sketch out some mock ups of your screens, and sketch out how data flows through your app to get you a clear understanding up front.
