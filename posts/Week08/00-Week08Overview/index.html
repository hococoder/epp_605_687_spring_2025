Welcome to Week 8! This week you'll learn about the following:

  • Introduction to AVKit - learn how to display videos in your app, even with Picture in Picture
  • Introduction to In App Purchases - learn how to make some money beyond the initial purchase of your app
  • Programming Assignment 8 - in this assignment you'll add AVKit support to the ToDo app, and add an unlock to get rid of pesky ads!

There is NOT a Week8_Starter or Week8_Final project for this week; example code can be found in the EPExampleApp project in the resource bundle. There is, as usual, a starter project for the homework
