Welcome to Week 11! This week you'll learn more about what you can do with App Intents, including:

  • Using App Intents with Shortcuts - it's easy to turn your App Intents into shortcuts you can use anywhere in the OS
  • Using App Intents with the Action Button - the action button on either the Apple Pencil or the newer iPhones can also trigger App Intents.
  • Using App Intents with Siri - this has to be the biggest change, especially as Apple Intelligence matures - using your App Intents with Siri.

There is NOT a Week11_Starter or Week11_Final project for this week; example code can be found in the EPExampleApp project in the resource bundle. There is, as usual, a starter project for the homework
