Welcome to Week 11! This week you'll learn about some technologies that will give you feedback from your app - from logging messages to straight out asking the user for feedback. You'll also get an introduction to using Siri Shortcuts.

  • Introduction to Logging - iOS 13 and 14 introduced improvements to the logging system which helps put debug statements in the system console
  • Introduction to User Feedback - learn about the built in feedback mechanism to prompt users to leave a App Store rating
  • Introduction to Shortcuts - learn about Shortcuts, which allow you to interface with your app through Siri and Notification Center
  • Programming Assignment 11 - in this assignment you'll integrate shortcuts and feedback into the app.

There is NOT a Week11_Starter or Week11_Final project for this week; example code can be found in the EPExampleApp project in the resource bundle. There is, as usual, a starter project for the homework
